# Travel planner app

A travelling web app that gives you the weather details of the place you want to visit and save the trip's details.

## Getting started

Install the required plugins and loaders by running:

- `npm install`

Then to create the compiled files (dist folder), run:

- `npm run prod`

Finally run:

- `npm run start`

to start the server. Go to this [link](http://127.0.0.1:8081/) to interact with the app.

## Testing

To test the app, run the following:

- `npm run test`
